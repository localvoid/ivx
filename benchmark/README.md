# ivx Benchmarks

Install all dependencies.

```sh
$ yarn
```

Compile dependencies.

```sh
$ lerna run dist
```

Compile TypeScript.

```sh
$ npm run dist
```

Run basic benchmarks.

```sh
$ node ./dist/benchmark.js
```

Launch HTTP server.

```sh
$ npm run serve
```

Run web benchmarks.

```sh
$ ./benchmark.sh
```
